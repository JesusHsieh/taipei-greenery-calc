# Greenery Architect｜建築基地綠化法規檢核系統

> **v1.0.0** — 台北市 + 新北市雙城市法規完整檢核，Clinical Architect 介面設計

🌐 **線上版本**：https://jesushsieh.github.io/taipei-greenery-calc/

---

## 簡介

輸入植栽配置資料，系統自動計算並即時判斷是否符合各項法定綠化標準。  
支援台北市與新北市兩套法規體系，右側面板即時顯示綠覆率與檢核狀態。

---

## 支援法規

### 台北市

| 條文 | 內容 | 備註 |
|------|------|------|
| 第4條 | 建築基地分類（五類） | 綠覆率 / 綠容率 / 固碳當量基準隨類別自動調整 |
| 第5條 | 綠覆率、綠容率計算 | 等效綠覆面積彙整 |
| 第6條 | 固碳當量（TCO₂ vs TCO₂c） | 依建築基地綠化設計技術規範表1，含 α 修正係數 |
| 第7條 | 法定空地植栽配置 | 高/低遮蔭喬木、灌木、其他植栽、臨道路開放空間 |
| 第8條 | 立體綠化設施 | 覆土深度分三段折算 |
| 第9條 | 屋頂平臺綠化 | 灌木比例 ≥ 30%、屋頂綠覆率 ≥ 50% |
| 第10條 | 綠覆面積計算方式 | 各類植栽係數 |
| 第12條 | 透水鋪面 | 透水率 ≥ 50% |

> 法規依據：《台北市新建築物綠化實施規則》（民國115年1月1日施行）、《建築基地綠化設計技術規範》

### 新北市

| 條文 | 內容 | 備註 |
|------|------|------|
| 施行細則第43條 | 實設空地植栽面積 | 扣除無法綠化面積後 ≥ 1/2 |
| 審議原則第8條(一)-3 | 喬木配置需求 | 每 36m² 應種植一棵（附表一三分級） |
| 審議原則第8條(三)(四) | 綠覆率 ≥ 100% | 各類植栽含加權係數（灌木 ×1.5 等） |
| 施行細則第44條 | 屋頂綠能設施 | 都設會審議案屋頂 ≥ 1/2 需設置綠能（綠化＋太陽光電） |

> 法規依據：《新北市都市設計審議原則》（民國110年修正）、《都市計畫法新北市施行細則》

## 功能特色

- **即時計算**：輸入任一數值即時更新右側結果面板
- **通過 / 未通過 / 待確認** 三態顯示每個檢核項目
- **綠覆率進度條**：視覺化顯示距目標的差距
- **固碳量計算**：完整實作 TCO₂ = (ΣGi×Ai) × α，並與 TCO₂c 比較
- **第44條開關**：非都設會審議案可切換關閉第44條檢核
- **立體綠化彈性**：新北市屋頂植栽可選擇是否計入綠覆率

## 介面設計

採用 **Stitch "Clinical Architect"** 設計系統：

- **佈局**：左側導覽列（240px）＋ 8欄輸入區 ＋ 4欄即時結果面板
- **色系**：深青 `#2e6767` primary，層疊式 surface 背景，無邊框卡片
- **字型**：Public Sans（標題）＋ Inter（內文）
- **輸入框**：底線 accent 樣式，無邊框設計
- **響應式**：手機版底部導覽列，側欄自動隱藏
