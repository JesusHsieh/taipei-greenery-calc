# 建築基地綠化法規檢核系統

輸入植栽資料，自動計算並判斷是否符合法定綠化標準。

## 支援法規

| 城市 | 法規 | 施行日期 |
|------|------|----------|
| 台北市 | 新建築物綠化實施規則 | 民國 115 年 1 月 1 日 |
| 新北市 | 建築基地綠化自治條例 | 建置中 |

## 台北市檢核項目

| 條文 | 內容 |
|------|------|
| 第4條 | 建築基地分類（五類） |
| 第5條 | 綠覆率、綠容率計算 |
| 第6條 | 固碳當量（係數待技術規範確認） |
| 第7條 | 法定空地植栽配置（含臨道路） |
| 第8條 | 立體綠化（高／低遮蔭分開） |
| 第9條 | 屋頂平臺綠化 |
| 第10條 | 綠覆面積計算方式 |
| 第12條 | 透水鋪面 |

## 本地開發

```bash
npm install
npm run dev
```

開啟 http://localhost:5173
